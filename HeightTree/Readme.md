# Динамическое программирование, задача посчитать высоту дерева

    Правила составления корневого дерева:
        - На вход подается массив переходов между узлами дерева: [4, -1, 4, 1, 1, 2, 5]
        - -1: корень дерева

### Пример: 
                t[1] = -1
               /         \
          t[3] = 1     t[4] = 1
                        /     \
                  t[0] = 4   t[2] = 4
                                 |
                             t[5] = 2
                                 |
                             t[6] = 5

        Высота для такого дерева будет = 4
